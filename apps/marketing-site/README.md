# PlaidHealth Marketing Site

Public-facing marketing website for PlaidHealth - The Plaid for Healthcare API.

## Features

### Landing Page
- Hero section with animated code preview
- Key metrics (integrations, patient records, uptime)
- Trusted by section with company logos
- Data connections overview (EMRs, Payers, Labs)
- Platform features grid
- Integration steps (how it works)
- Customer testimonials
- Interactive API playground with live code examples
- Security deep-dive section
- Pricing tiers
- Call-to-action sections

### Additional Pages
- `/pricing` - Detailed pricing with feature comparison
- `/use-cases` - Industry-specific solutions
- `/docs` - Documentation overview
- `/about` - Company information
- `/contact` - Contact form and demo requests
- `/blog` - Company blog (placeholder)

### Components
- Responsive header with mobile menu
- Comprehensive footer with navigation
- Animated sections using Framer Motion
- Interactive API playground with tab switching
- Copy-to-clipboard code examples

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- Lucide React icons

## Development

```bash
# From repository root
npm install
npm run dev:marketing

# Or from this directory
npm install
npm run dev
```

Runs on http://localhost:3002
